Inside TrueFocus

TrueFocus is built to improve academic integrity while protecting user privacy. This page explains exactly how the system works and why it is different.

How It Works

Camera Input

The system accesses your webcam locally and processes frames in real time.

AI Detection

Head movement and eye direction are analyzed using computer vision models.

Behavior Analysis

If suspicious behavior is detected, alerts are generated instantly.

Privacy Architecture

100% Local Processing

No data is uploaded to any server. Everything runs directly on the device.

No Recording

Video footage is never saved, stored, or streamed.

No Biometric Storage

The system does not store facial data or identity information.

Technology Stack

Computer Vision

Uses real-time detection algorithms for accurate behavioral monitoring.

Lightweight Engine

Optimized to run efficiently even on mid-range systems.

Cross-Platform Ready

Currently available for Windows. macOS and Linux coming soon.
